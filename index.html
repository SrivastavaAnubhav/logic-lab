Logic lab

 

Instructions: Make a well-formed formula according to the following rules. Note that the program is case-sensitive (e.g. use OR, not or).

  • A string whose name is not AND, OR, NOT, IMP, or IFF and which does not contain a ( or a ) is a well-formed formula.
  • If A is a well-formed formula then (NOT A) is a well-formed formula.
  • If A is a well-formed formula and B is a well-formed formula then:
    • (A AND B) is a well-formed formula.
    • (A OR B) is a well-formed formula.
    • (A IMP B) is a well-formed formula (if/implication).
    • (A IFF B) is a well-formed formula (if and only if).
